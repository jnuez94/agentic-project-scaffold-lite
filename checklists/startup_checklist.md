# Startup Checklist

Use this before agents begin coordinated work.

- [ ] Project goal is written.
- [ ] Near-term deliverable is written.
- [ ] Hard boundaries are written.
- [ ] Coordination substrate is selected.
- [ ] Canonical status model is configured.
- [ ] Agent profiles exist.
- [ ] Decision rights are assigned.
- [ ] Sensitive-data rules are defined.
- [ ] Initial tasks are created.
- [ ] Review flow is defined.
- [ ] External sharing rules are defined.
- [ ] Release authority is defined.
- [ ] Every active agent knows to sync before starting work.
- [ ] Every active agent can create tasks for self or others when needed.
